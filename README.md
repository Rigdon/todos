# Todos Project

A task management application built with a Go backend and TypeScript frontend.

## Technology Stack

- **Backend**: Go (Module: `todos`)
- **Frontend**: TypeScript
- **Database**: SQLite3
- **Configuration**: OpenSpec for AI-driven development

## Getting Started

### Backend
```bash
go mod tidy
go run .
```

### Frontend
```bash
npm install
npm run build
```
